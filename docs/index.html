Features

Everything your tech team needs

Live verse detection

Three detection engines — direct reference parsing, verbatim phrase matching, and fingerprint coverage — vote and confirm in real time as the sermon plays out.

Reading mode

Detects when a preacher is reading through a passage and tightens the verse-advance sensitivity automatically, so consecutive verses advance smoothly without gaps.

ProPresenter integration

Pushes detected verses directly to ProPresenter's live output via the REST API. Works alongside your existing slide deck with zero disruption.

OBS WebSocket

Writes verse text to any OBS text source over WebSocket. Works with streaming, recordings, or a confidence monitor — no plugins required.

Theme Studio

Design layered on-screen looks — background, verse text, reference — with custom fonts, colors, gradients, and blur. Live preview before pushing to the display.

Sermon context memory

Remembers the book and chapter being preached so bare references like "verse 17" or "the next verse" resolve correctly without a full citation.

Setup

Up and running in 5 minutes

1

Download & install

Download the installer for your platform above, open it, and drag KAIRO to your Applications folder (macOS) or run the installer (Windows).

2

Get a free Deepgram API key

Sign up at deepgram.com — free tier includes 12,000 minutes/month, more than enough for weekly services. Paste the key into KAIRO Settings → Audio.

3

Connect your audio

Plug in a line-level feed from your mixing board, or use a USB interface. Select the audio input in Settings → Audio Device. A direct line-in gives the cleanest transcription.

4

Hit Start Listening

KAIRO begins transcribing and displays detected verses in the live queue. From there they push to ProPresenter, OBS, or an external display — automatically.

Requirements

What you need

Required

  • Deepgram API key (free at deepgram.com)
  • Audio input — line-in from board or USB interface
  • macOS 12+, Windows 10+, or Linux (Ubuntu 22+)
  • Internet connection for transcription

Optional integrations

  • ProPresenter 7 (REST API on same network)
  • OBS Studio (WebSocket plugin built-in since v28)
  • Syphon-compatible app (macOS only)
  • Secondary display / confidence monitor
